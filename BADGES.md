## Custom Badge Services

### Shields.io

Create custom badges at https://shields.io/:

```
https://img.shields.io/badge/your-badge-blue?style=flat-square
```

### Badge Generator

Use https://badgen.net/ for alternative badge styles:

```
https://badgen.net/badge/your/badge/blue
```
