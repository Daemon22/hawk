Hawk

Device Detection & Environment Monitoring

What is Hawk?

Hawk is a comprehensive device detection and environment monitoring library that scans and identifies your device, operating system, and runtime environment in real-time.

Features

  • Cross-platform device detection (mobile, desktop, tablet, terminal)
  • Operating system identification with version detection
  • CPU architecture and hardware capability detection
  • Runtime environment identification (browser, Node.js, Electron, etc.)
  • Device capability detection (WebGL, geolocation, camera, etc.)
  • Metadata collection (timezone, language, hardware concurrency)
  • Built-in caching for performance optimization
  • Modular and extensible architecture

Installation

Browser

<script src="path/to/hawk.js"></script>
<script>
  const monitor = new DeviceMonitor();
  monitor.detect().then(profile => {
    console.log(profile);
  });
</script>

Node.js

const DeviceMonitor = require("./path/to/hawk.js");

async function getDeviceProfile() {
  const monitor = new DeviceMonitor({ debugMode: true });
  const profile = await monitor.detect();
  console.log(profile);
}

getDeviceProfile();

Basic Usage

const monitor = new DeviceMonitor();

monitor.detect().then(profile => {
  console.log("Device Profile:", profile);
});

// With options
const monitor = new DeviceMonitor({
  enableCache: true,
  cacheTTL: 600000,  // 10 minutes
  debugMode: true
});

// Clear cache
monitor.clearCache();

API Reference

DeviceMonitor Class

constructor(options) - Initialize with optional configuration

detect() - Asynchronously detect device profile (returns Promise)

clearCache() - Clear cached detection results

Configuration Options

Option Type Default Description
enableCache boolean true Enable/disable result caching
cacheTTL number 300000 Cache TTL in milliseconds
debugMode boolean false Enable console logging
customDetectors Array [] Custom detector instances

Device Profile Output

The detect() method returns a comprehensive profile with the following structure:

{
  timestamp: "2024-01-15T10:30:00.000Z",
  os: { name, version, platform, family, kernel, distribution },
  deviceType: { type, isMobile, isDesktop, isTablet, ... },
  screen: { width, height, pixelRatio, colorDepth, orientation, ... },
  architecture: { cpu, bits, endian, cores },
  environment: { runtime, runtimeVersion, isBrowser, isNode, ... },
  capabilities: { webgl, serviceWorker, geolocation, camera, ... },
  userAgent: "...",
  metadata: { timezone, language, hardwareConcurrency, ... }
}

Examples

Conditional Logic Based on Device Type

monitor.detect().then(profile => {
  if (profile.deviceType.isMobile) {
    console.log('Loading mobile version');
  } else if (profile.deviceType.isTablet) {
    console.log('Loading tablet version');
  } else {
    console.log('Loading desktop version');
  }
});

Checking Device Capabilities

monitor.detect().then(profile => {
  if (profile.capabilities.geolocation) {
    navigator.geolocation.getCurrentPosition(position => {
      console.log('User location:', position.coords);
    });
  }
  
  if (profile.capabilities.camera) {
    console.log('Camera is available');
  }
});
